log()

Adds lines to the results file reporting columns specific to the controller

See each controller's documentation for details about logging. Any controller-specific columns will be added after the "Comment" column, and the "Value" column will read "See extra columns". The timestamp in the EventTime column corresponds to when the controller finished.

Source:

Example
newController("Question", {q: "Is this a question?", as:["Yes","No"]}).log().print().wait()
// ...,PennElementName,PennElementType,Parameter,Value,EventTime,Comments,Question.NULL.if.none,Answer,Whether.or.not.answer.was.correct,Time.taken.to.answer
// ...,Question,Controller,Question,See extra columns,1688484096570,4 extra columns added for this controller,NULL,Yes,NULL,4579
