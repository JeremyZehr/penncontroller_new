jump(predicate)

Finds the first trial in the sequence with a corresponding label and will continue the sequence from there

Note that this command does not end the current trial. If the trial found comes before the current one, then you will effectively jump back in time in the sequence of trials; if it comes later, then you will skip any trials between this one and that one.

Parameters:
Name Type Description
predicate string | function

A string representing a label, or a predicate matching the label of a trial you want to go to next
Source:
  • standardCommands.js, line 689
Example
// Places the first trial labeled `"experiment"` in the sequence next, and immediately ends this one, upon clicking the button
newButton("Go to the experiment").callback( jump("experiment"),end() ).print()

Type Definitions

Command()

PennController commands are called on element references via newElement() or getElement(), where Element stands for the element type. Underlyingly, commands are (asynchronous) functions and you can pass javascript functions wherever PennController commands are referenced

Source:
  • element.js, line 78
Example

Commands in a callback

// Print a Text element on the page and a message in the console when the button is clicked
newButton("Click me!")
  .callback(
    newText("You clicked the button!").print(), // first command
    ()=>console.log("click") // second command
  )

Coordinate

A number (in px) or a string unit that can start with "left at", "center at", "right at", "top at", "middle at" or "bottom at". When the string only contains a unit value, it implicitly starts with "left at" or "top at", as applicable

  • number | string
