center()

Horizontally centers the element.

More precisely, the element's main DOM node is centered relative to its container, while the container's width is set to 100%: the element therefore appears centered relative to its container's parent.

left()

Aligns the element to the left.

The left edge of the element's main DOM node will be aligned relative to its container, while the container's width is set to 100%: the element therefore appears aligned with the left edge of its container's parent.

Source:
  • standardCommands.js, line 464
Example

  .add("center at 25%", 0, newButton("To the left").callback(getText("Hello world").left()) )
  .add("center at 75%", 0, newButton("To the right").callback(getText("Hello world").right()) )
  .center()
  .print()

Examples

  .wait()

yOrTarget number | string <optional>

The new height of the element, or "parent" if you want to apply the scaling factor wOrFactor to the element's container
target string <optional>

Pass "parent" when also setting wOrFactor and hOrTarget to new width and height to the element's container
